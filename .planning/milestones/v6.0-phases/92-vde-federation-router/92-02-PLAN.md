**CatalogEntry.cs** -- the universal catalog record shared by all 4 hierarchy levels:
- Namespace: `DataWarehouse.SDK.VirtualDiskEngine.Federation`
- `[SdkCompatibility("6.0.0", Notes = "Phase 92: VDE 2.0B Shard Catalog (VFED-06)")]`
- `public readonly record struct CatalogEntry`
- Fields:
  - `Guid ShardVdeId` -- the VDE instance this entry routes to
  - `int StartSlot` -- beginning of the hash-slot range (inclusive)
  - `int EndSlot` -- end of the hash-slot range (exclusive)
  - `ShardLevel Level` -- which catalog level this entry represents (Root/Domain/Index/Data)
  - `CatalogEntryStatus Status` -- Active/Migrating/Draining/Offline
  - `long BloomFilterDigest` -- 8-byte bloom filter fingerprint for quick negative rejection (0 = no filter)
  - `long CreatedUtcTicks` -- creation timestamp
  - `long LastAccessedUtcTicks` -- last access timestamp for LRU decisions
- Enum `CatalogEntryStatus : byte { Active = 0, Migrating = 1, Draining = 2, Offline = 3 }`
- Binary serialization: 64 bytes total. Layout:
  - [0..15] ShardVdeId (16 bytes GUID)
  - [16..19] StartSlot (int32 LE)
  - [20..23] EndSlot (int32 LE)
  - [24] Level (byte)
  - [25] Status (byte)
  - [26..27] Reserved (2 bytes, zero)
  - [28..35] BloomFilterDigest (int64 LE)
  - [36..43] CreatedUtcTicks (int64 LE)
  - [44..51] LastAccessedUtcTicks (int64 LE)
  - [52..63] Reserved (12 bytes, zero, future use)
- Methods: `WriteTo(Span<byte> dest)`, `static ReadFrom(ReadOnlySpan<byte> src)` using BinaryPrimitives
- `public const int SerializedSize = 64;`
- `public bool ContainsSlot(int slot) => slot >= StartSlot && slot < EndSlot;`
- `public bool IsActive => Status == CatalogEntryStatus.Active;`

**RootCatalog.cs** -- Level 0 catalog, fully in-memory:
- `[SdkCompatibility("6.0.0", Notes = "Phase 92: VDE 2.0B Root Catalog (VFED-08)")]`
- `public sealed class RootCatalog`
- Constructor: `RootCatalog(CatalogReplicationConfig replicationConfig, int maxEntries = 1_048_576)`
- Internal storage: `private CatalogEntry[] _entries` sorted by StartSlot. Use array (not dictionary) for cache-friendly binary search.
- `private int _count` -- active entry count
- `private readonly ReaderWriterLockSlim _lock = new()`
- Methods:
  - `public CatalogEntry? LookupDomain(int domainSlot)` -- binary search _entries by ContainsSlot, O(log n). Returns the entry whose [StartSlot, EndSlot) range contains domainSlot, or null.
  - `public void AddEntry(CatalogEntry entry)` -- insert maintaining sorted order. Validate no overlapping ranges. Throw if _count >= maxEntries.
  - `public bool RemoveEntry(Guid shardVdeId)` -- remove entry by VDE ID, compact array.
  - `public void UpdateEntry(CatalogEntry updated)` -- find by VdeId, replace in-place.
  - `public IReadOnlyList<CatalogEntry> GetAllEntries()` -- snapshot copy for admin view.
  - `public int EntryCount => _count;`
  - `public long EstimatedMemoryBytes => (long)_count * CatalogEntry.SerializedSize + 64;` -- for monitoring against 100MB limit
- Serialization for Raft log replication:
  - `public void SerializeTo(Stream stream)` -- write count (int32 LE) then each entry (64 bytes)
  - `public static RootCatalog DeserializeFrom(Stream stream, CatalogReplicationConfig config)` -- read back
- Thread safety: ReaderWriterLockSlim (read lock for LookupDomain, write lock for Add/Remove/Update)
