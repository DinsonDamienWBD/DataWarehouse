     a. KernelOffset = PreambleHeader.SerializedSize (64)
     b. SpdkOffset = KernelOffset + kernelSizeBytes
     c. If !profile.IncludeSpdk: SpdkOffset = KernelOffset, spdkSizeBytes = 0 (skip SPDK)
     d. RuntimeOffset = SpdkOffset + spdkSizeBytes
     e. ContentHashOffset = RuntimeOffset + runtimeSizeBytes (32B BLAKE3 follows runtime)
     f. PreambleTotalSize = ContentHashOffset + 32 (hash size)
     g. If profile.SignWithEd25519: PreambleTotalSize += 64 (Ed25519 signature)
     h. VdeOffset = ceil(PreambleTotalSize / 4096) * 4096 (4KiB-aligned)
     i. Validate all uint32 offsets don't overflow (< uint.MaxValue)
     j. Returns `PreambleLayoutResult` with all offsets + total sizes
