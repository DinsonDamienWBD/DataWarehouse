**MetricsLogWriter.cs** -- class with `[SdkCompatibility("6.0.0")]`:
- Implements the MLOG region header (38 bytes per spec):
  - `RegionMagic` (8 bytes): `0x4D45545249435300` ("METRICS\0")
  - `EntryCount` (8 bytes): uint64 LE total metric entries
  - `OldestTimestamp` (8 bytes): uint64 LE UTC nanoseconds
  - `NewestTimestamp` (8 bytes): uint64 LE UTC nanoseconds
  - `MetricIdCount` (2 bytes): uint16 LE distinct metric IDs
  - `RetentionSeconds` (4 bytes): uint32 LE retention period
  - `DownsampleFactor` (2 bytes): uint16 LE downsampling factor
- Constants: `HeaderSize = 38`, `EntrySize = 18`, `RegionMagic = 0x4D45545249435300UL`
- Constructor: `MetricsLogWriter(uint retentionSeconds = 86400, ushort downsampleFactor = 60)`

Metric entry format (18 bytes per spec):
- `Timestamp` (8 bytes): uint64 LE UTC nanoseconds
- `MetricId` (2 bytes): uint16 LE
- `Value` (8 bytes): int64 LE or float64 (depends on MetricId)

Core methods:
- `void AppendEntry(ulong timestampNanos, ushort metricId, long intValue)` -- appends int64 entry
- `void AppendEntryFloat(ulong timestampNanos, ushort metricId, double floatValue)` -- appends float64 entry (uses BitConverter.DoubleToInt64Bits for binary-compatible storage)
- `ReadOnlySpan<byte> SerializeRegion(Span<byte> buffer, int blockSize)` -- serializes header + all entries into MLOG blocks with UniversalBlockTrailer on each block using `BlockTypeTags.MLOG`
- `static MetricsLogWriter DeserializeRegion(ReadOnlySpan<byte> buffer, int blockSize, int blockCount)` -- reads header, validates RegionMagic, reads entries
- `int RequiredBlocks(int blockSize)` -- computes blocks needed
- `void RunCompaction()` -- groups entries older than RetentionSeconds by MetricId in windows of DownsampleFactor entries, replaces each window with averaged entry, frees compacted entries
- `IReadOnlyList<(ulong Timestamp, ushort MetricId, long RawValue)> GetEntries()` -- returns all entries
- `int EntryCount` property

Track distinct MetricIds via a `HashSet<ushort>` updated on each append. Track oldest/newest timestamps.

Use `BinaryPrimitives` for all serialization. Use `System.Buffers.Binary` namespace.
