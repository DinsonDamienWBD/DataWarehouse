Create `DeltaExtentModule` readonly struct with `[SdkCompatibility("6.0.0")]` in namespace `DataWarehouse.SDK.VirtualDiskEngine.Format`:

Per spec: 8B at inode fixed-offset 0x1F2 in DataOsModuleArea: [MaxDeltaDepth:2][CurrentDepth:2][CompactionPolicy:4]

- `ushort MaxDeltaDepth` -- maximum delta chain depth before compaction (default 8)
- `ushort CurrentDepth` -- current depth of delta chain for this inode
- `DeltaCompactionPolicy CompactionPolicy` -- enum: Eager=0 (compact at MaxDeltaDepth), Lazy=1 (compact at 2x MaxDeltaDepth), OnRead=2 (compact on next read), Manual=3 (never auto-compact)
- `const int SerializedSize = 8`
- `const byte ModuleBitPosition = 22` -- DELT in ModuleManifest
- `const int InodeOffset = 0x1F2` -- fixed offset in 512B inode DataOsModuleArea (byte 498)
- `bool NeedsCompaction` -- true when CurrentDepth > MaxDeltaDepth (for Eager policy)
- `DeltaExtentModule WithIncrementedDepth()` -- returns new module with CurrentDepth + 1
- `DeltaExtentModule WithResetDepth()` -- returns new module with CurrentDepth = 0 (after compaction)
- `static void Serialize(in DeltaExtentModule module, Span<byte> buffer)` -- 8B LE
- `static DeltaExtentModule Deserialize(ReadOnlySpan<byte> buffer)` -- 8B LE
- `static DeltaExtentModule Default` -- MaxDeltaDepth=8, CurrentDepth=0, Policy=Eager

Create `DeltaExtentPatcher` class with `[SdkCompatibility("6.0.0")]` in namespace `DataWarehouse.SDK.VirtualDiskEngine.CopyOnWrite`:

VCDIFF-style binary patch engine for sub-block delta extents:

- Constructor: `int blockSize`
- `DeltaPatchResult GeneratePatch(ReadOnlySpan<byte> originalBlock, ReadOnlySpan<byte> modifiedBlock)` -- generates VCDIFF-style patch. Returns patch data + whether delta is worth it (patch size < threshold percentage of block).
  - `DeltaPatchResult` readonly struct: `byte[] PatchData`, `int PatchSize`, `bool IsDeltaWorthwhile` (true if PatchSize < blockSize * 0.5), `double CompressionRatio`
- `byte[] ApplyPatch(ReadOnlySpan<byte> baseBlock, ReadOnlySpan<byte> patchData)` -- applies patch to base block, returns reconstructed block
- `byte[] ApplyDeltaChain(ReadOnlySpan<byte> baseBlock, IReadOnlyList<byte[]> patches)` -- applies chain of patches sequentially, returns final block
- `byte[] FlattenChain(ReadOnlySpan<byte> baseBlock, IReadOnlyList<byte[]> patches)` -- same as ApplyDeltaChain but used for compaction (produces new base block)
- `bool ShouldUseDelta(ReadOnlySpan<byte> originalBlock, ReadOnlySpan<byte> modifiedBlock, double threshold = 0.1)` -- returns true if modification is less than threshold fraction of block

VCDIFF patch format (simplified, internal):
  [PatchHeaderMagic:4 = 0x44454C54 "DELT"][OriginalSize:4][PatchedSize:4][NumOps:2]
  followed by operations: [OpType:1 (Copy=0, Insert=1)][Offset:4][Length:2][Data:variable for Insert]

Use `System.Buffers.Binary.BinaryPrimitives` for serialization. Use `DataWarehouse.SDK.Contracts` for SdkCompatibility.
