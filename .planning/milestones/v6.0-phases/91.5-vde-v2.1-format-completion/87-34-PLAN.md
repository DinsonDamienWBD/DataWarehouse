Epoch index:
- `EpochEntry` readonly struct: `long SnapshotId, ulong Epoch (UTC nanoseconds), uint GenerationNumber`
- `List<EpochEntry> _epochIndex` -- sorted by Epoch ascending, built from Snapshot Table at mount
- `Task BuildEpochIndexAsync(CancellationToken ct)` -- scan Snapshot Table region, build sorted list

Point query:
- `Task<TemporalQueryResult> QueryAtEpochAsync(long inodeNumber, ulong epochNanoseconds, CancellationToken ct)`:
  1. Binary search _epochIndex for largest epoch <= requested (O(log N))
  2. Identify the snapshot active at that epoch
  3. Read inode state from that snapshot's inode table copy
  4. Resolve extent pointers valid at that generation
  5. Return TemporalQueryResult with resolved extents

- `Task<TemporalQueryResult> QueryAtTimestampAsync(long inodeNumber, DateTimeOffset timestamp, CancellationToken ct)`:
  - Convert timestamp to UTC nanoseconds, delegate to QueryAtEpochAsync

- `TemporalQueryResult` readonly struct:
  - `long InodeNumber, ulong ResolvedEpoch, long SnapshotId`
  - `long FileSize` -- size at that point in time
  - `IReadOnlyList<InodeExtent> Extents` -- extent pointers valid at epoch
  - `bool InodeExisted` -- false if inode didn't exist at requested time
  - `DateTimeOffset ResolvedTimestamp`

Range query:
- `Task<IReadOnlyList<TemporalVersion>> GetVersionHistoryAsync(long inodeNumber, ulong fromEpoch, ulong toEpoch, CancellationToken ct)`:
  - Returns all version changes within epoch range
  - `TemporalVersion` readonly struct: `ulong Epoch, long FileSize, int ExtentCount, uint GenerationNumber`
