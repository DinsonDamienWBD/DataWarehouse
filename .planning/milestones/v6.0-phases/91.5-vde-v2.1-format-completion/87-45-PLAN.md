1. Create `VdeDetector.cs` with `[SdkCompatibility("6.0.0")]`:
   - `enum VdeFileType { PreamblePresent, V2PreambleFree, LegacyV1, NotDwvd }`
   - `static async Task<VdeDetectionResult> DetectAsync(Stream stream, CancellationToken ct)`:
     a. Read first 8 bytes from stream position 0
     b. If bytes 0-7 match "DWVD-BOO" magic → return PreamblePresent with VdeOffset from header
     c. If bytes 0-3 match "DWVD" (0x44575644 big-endian) → read byte 4 (FormatMajor):
        - If major >= 2 → return V2PreambleFree with VdeOffset=0
        - Else → return LegacyV1 with VdeOffset=0
     d. Else → return NotDwvd
   - `readonly struct VdeDetectionResult` with VdeFileType, long VdeOffset, PreambleHeader? Preamble (nullable, only set when PreamblePresent)

2. Create `PreambleReader.cs` with `[SdkCompatibility("6.0.0")]`:
   - `static PreambleHeader ReadHeader(ReadOnlySpan<byte> buffer)` — deserializes 64 bytes, validates magic, validates header checksum (compute BLAKE3 of bytes[0..55], truncate to 8 bytes, compare to HeaderChecksum field). Throws `InvalidOperationException` if magic invalid. Throws `DataCorruptionException` (or InvalidOperationException) if checksum mismatch.
   - `static async Task<PreambleHeader> ReadHeaderAsync(Stream stream, CancellationToken ct)` — reads 64 bytes from position 0, delegates to ReadHeader.
   - `static bool TryValidateContentHash(Stream stream, PreambleHeader header)` — reads kernel+spdk+runtime payload bytes, computes BLAKE3, compares to the 32-byte hash at ContentHashOffset. Returns true if match. This validates all executable payloads are intact.
   - Use `System.IO.Hashing` for BLAKE3 if available, otherwise use a simple placeholder that computes using SHA256 with a TODO comment noting BLAKE3 should be used (BLAKE3 NuGet package may not be referenced yet — check and use whatever hash is available in the SDK, with appropriate documentation).

3. Create `PreambleWriter.cs` with `[SdkCompatibility("6.0.0")]`:
   - `static void WriteHeader(in PreambleHeader header, Span<byte> buffer)` — serializes header, computes BLAKE3(bytes[0..55]) truncated to 8 bytes, writes to HeaderChecksum position (bytes 56-63), then re-serializes with the correct checksum.
   - `static async Task WriteHeaderAsync(Stream stream, PreambleHeader header, CancellationToken ct)` — writes 64 bytes at position 0.
   - `static PreambleHeader CreateHeader(ulong vdeOffset, uint kernelOffset, uint kernelSize, uint spdkOffset, uint spdkSize, uint runtimeOffset, uint runtimeSize, PreambleFlags flags, TargetArchitecture arch)`:
     a. Validates VdeOffset is 4KiB-aligned
     b. Computes PreambleTotalSize = max(kernelOffset+kernelSize, spdkOffset+spdkSize, runtimeOffset+runtimeSize)
     c. Validates VdeOffset >= PreambleTotalSize
     d. Builds PreambleHeader with PreambleVersion=1, computed Flags from flags+arch, ContentHashOffset computed as runtimeOffset+runtimeSize (content hash immediately follows runtime)
     e. Returns header (HeaderChecksum filled in by WriteHeader)
