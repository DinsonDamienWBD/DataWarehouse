<objective>
Implement Epoch-to-ZNS Hardware Symbiosis (ZNSM, Module bit 23, VOPT-34): ZNS zone map region and zone-aware allocator that maps MVCC epochs 1:1 to physical ZNS zones for elimination of SSD garbage collection.

Purpose: On ZNS NVMe drives, map each MVCC epoch to a physical zone. When an epoch dies, issue a single ZNS_ZONE_RESET hardware command instead of millions of per-block TRIMs. Write latency drops from 500us to 15us, SSD lifespan increases ~300%.
Output: Two files in `DataWarehouse.SDK/VirtualDiskEngine/`.
</objective>

Per spec: zone map entry is 16B: [EpochID:8][ZoneID:4][State:2][Flags:2]

- `ZnsZoneMapEntry` readonly struct (16 bytes):
  - `ulong EpochId` -- MVCC epoch mapped to this zone
  - `uint ZoneId` -- physical ZNS zone identifier
  - `ZnsZoneState State` -- enum: Free=0, Active=1, Full=2, Dead=3, Resetting=4
  - `ZnsZoneFlags Flags` -- [Flags] enum: None=0, MetadataZone=1, DataOnly=2, Reserved=4
  - `const int EntrySize = 16`
  - `static void Serialize(in ZnsZoneMapEntry entry, Span<byte> buffer)` -- 16B LE
  - `static ZnsZoneMapEntry Deserialize(ReadOnlySpan<byte> buffer)` -- 16B LE

- `ZnsZoneMapRegion` class:
  - Constructor: `long regionStartBlock, int blockSize`
  - `const uint BlockTypeTag = 0x5A4E534D` -- "ZNSM" matching BlockTypeTags
  - `int MaxEntriesPerBlock` -- computed: (blockSize - FormatConstants.UniversalBlockTrailerSize) / ZnsZoneMapEntry.EntrySize
  - `List<ZnsZoneMapEntry> Entries { get; }` -- all zone map entries
  - `void AddEntry(ZnsZoneMapEntry entry)`
  - `ZnsZoneMapEntry? FindByEpoch(ulong epochId)` -- lookup by epoch
  - `ZnsZoneMapEntry? FindByZone(uint zoneId)` -- lookup by zone
  - `IReadOnlyList<ZnsZoneMapEntry> GetDeadZones()` -- entries with State=Dead
  - `void MarkZoneDead(ulong epochId)` -- transitions Active/Full to Dead
  - `void MarkZoneReset(uint zoneId)` -- transitions Dead to Free
  - `int Serialize(Span<byte> buffer)` -- writes all entries, returns bytes written
  - `static ZnsZoneMapRegion Deserialize(ReadOnlySpan<byte> buffer, long regionStartBlock, int blockSize)` -- reads entries
