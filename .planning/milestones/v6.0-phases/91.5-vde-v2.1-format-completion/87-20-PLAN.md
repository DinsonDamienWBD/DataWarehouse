Create `VolatileKeyRingEntry` readonly record struct with `[SdkCompatibility("6.0.0")]`:
- `Guid EphemeralKeyId` -- 16-byte unique key identifier
- `byte[] KeyMaterial` -- the actual encryption key (32 bytes for AES-256)
- `ulong TtlEpoch` -- MVCC epoch at which key expires
- `uint SlotIndex` -- position in the volatile ring
- `DateTimeOffset CreatedUtc` -- creation timestamp
- `bool IsExpired(ulong currentEpoch)` -- returns true if currentEpoch >= TtlEpoch

Create `IVolatileKeyRing` interface with `[SdkCompatibility("6.0.0")]`:
- `Task<VolatileKeyRingEntry> AllocateKeyAsync(ulong ttlEpoch, CancellationToken ct)` -- generates new ephemeral key, stores in volatile ring, returns entry
- `Task<VolatileKeyRingEntry?> GetKeyAsync(Guid keyId, CancellationToken ct)` -- retrieves key by ID, returns null if expired/dropped
- `Task<bool> DropKeyAsync(Guid keyId, CancellationToken ct)` -- O(1) crypto-shred: removes key from ring, returns true if found
- `Task<int> ReapExpiredAsync(ulong currentEpoch, CancellationToken ct)` -- removes all keys with TtlEpoch <= currentEpoch, returns count reaped
- `int ActiveKeyCount { get; }` -- number of live keys in ring
- `int Capacity { get; }` -- maximum slots in ring
- NOTE: Keys MUST NOT be persisted to VDE blocks. Implementations: RAM-only (ConcurrentDictionary) or TPM-sealed (via Tpm2Provider). Anti-pattern: no KMS backup, no disk serialization.

Create `EphemeralKeyModule` readonly struct with `[SdkCompatibility("6.0.0")]`:
- Per VDE spec: 32 bytes in Module Overflow Block, packed as [EphemeralKeyID:16][TTL_Epoch:8][KeyRingSlot:4][Flags:4]
- Properties: `Guid EphemeralKeyId`, `ulong TtlEpoch`, `uint KeyRingSlot`, `EphemeralKeyFlags Flags`
- `EphemeralKeyFlags` [Flags] enum: None=0, TpmSealed=1, AutoReap=2, ForceExpireOnUnmount=4
- `const int SerializedSize = 32`
- `const byte ModuleBitPosition = 20` -- corresponds to EKEY in ModuleManifest
- `static void Serialize(in EphemeralKeyModule module, Span<byte> buffer)` -- writes 32 bytes LE
- `static EphemeralKeyModule Deserialize(ReadOnlySpan<byte> buffer)` -- reads 32 bytes LE
- `bool IsExpired(ulong currentEpoch)` -- checks TtlEpoch

All classes in namespace `DataWarehouse.SDK.VirtualDiskEngine.Encryption`.
