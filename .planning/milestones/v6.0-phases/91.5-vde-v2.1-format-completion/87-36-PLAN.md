Core scan:
- `Task<TagScanResult> ScanAsync(IReadOnlyList<TagPredicate> predicates, long inodeTableStartBlock, long inodeCount, CancellationToken ct)`:
  1. Calculate blocks to read: ceil(inodeCount / InodesPerBlock)
  2. For each batch of BatchSizeBlocks:
     a. Read block batch into buffer
     b. For each inode in batch (stride = 512B):
        - Extract InlineTagCount at offset 312 (uint32)
        - If InlineTagCount == 0: skip inode
        - Extract 176B InlineTagArea starting at byte offset 320 within inode
        - For each tag slot (32B each, up to InlineTagCount, max 5):
          Parse: [NamespaceHash:4][NameHash:4][ValueType:1][ValueLen:1][Value:<=22B]
        - If UseSimdAcceleration: batch-compare NamespaceHash+NameHash across slots using Vector256
        - Evaluate all predicates against parsed tags
        - If all predicates match: add inode number to results
  3. Return TagScanResult

  - Return true if any lane matches
