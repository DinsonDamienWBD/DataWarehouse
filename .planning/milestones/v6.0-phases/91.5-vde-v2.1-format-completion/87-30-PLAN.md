Create `DedupConfig` class with `[SdkCompatibility("6.0.0")]` in CopyOnWrite namespace:
- `int MinExtentBlockCount { get; set; } = 4` -- skip tiny extents (below 16KB at 4KB blocks)
- `int MaxConcurrentScans { get; set; } = 2` -- limit parallel dedup I/O
- `long MaxBytesPerPass { get; set; } = 1L * 1024 * 1024 * 1024` -- 1GB budget per pass
- `bool VerifyBeforeShare { get; set; } = true` -- byte-compare before trusting hash match
- `TimeSpan ScanInterval { get; set; } = TimeSpan.FromMinutes(30)`

Create `ContentAddressableDedup` class with `[SdkCompatibility("6.0.0")]`:
- Constructor: `IBlockDevice device, IBlockAllocator allocator, DedupConfig config, int blockSize`
- Internal: `Dictionary<Hash128, ExtentRef> _hashIndex` mapping ExpectedHash -> first extent reference
- `ExtentRef` readonly struct: `long InodeNumber, int ExtentIndex, long StartBlock, int BlockCount`

Core operations:
- `Task<DedupResult> ScanAndDedupAsync(IEnumerable<(long InodeNumber, InodeExtent[] Extents)> inodes, CancellationToken ct)`:
  1. Build hash index from all extent ExpectedHash values (skip extents with zero hash or already Shared_COW)
  2. Identify duplicate groups (same ExpectedHash, different StartBlock)
  3. For each duplicate group:
     a. Pick canonical extent (lowest StartBlock = keeper)
     b. If VerifyBeforeShare: read and byte-compare keeper vs duplicate blocks
     c. Increment reference count on keeper's blocks
     d. Set Shared_COW flag on both keeper and duplicate extent pointers
     e. Update duplicate extent's StartBlock to point to keeper's blocks
     f. Free duplicate's original blocks via allocator
  4. Track bytes saved, extents deduplicated

- `DedupResult` readonly struct:
  - `int ExtentsDeduplicated, long BlocksFreed, long BytesSaved`
  - `int DuplicateGroupsFound, int VerificationFailures`
  - `TimeSpan Duration`

- `Task<DedupStats> GetStatsAsync(CancellationToken ct)`:
  - `DedupStats` readonly struct: `long TotalExtentsScanned, long UniqueHashes, long DuplicateExtents, double DedupRatio`

Reference counting:
- `Dictionary<long, int> _refCounts` tracking block reference counts
- `void IncrementRef(long startBlock, int blockCount)` -- increment for shared blocks
- `bool TryDecrementRef(long startBlock, int blockCount)` -- decrement, return true if refcount hits zero (safe to free)
