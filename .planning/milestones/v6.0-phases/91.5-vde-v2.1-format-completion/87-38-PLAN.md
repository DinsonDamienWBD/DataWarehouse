- Constructor: `IBlockDevice device, MerkleIntegrityVerifier? merkleVerifier, int blockSize, IntegrityChainConfig config`

Level 1 - Block verification (XxHash64 from TRLR):
- `Task<BlockLevelResult> VerifyBlocksAsync(long dataRegionStart, InodeExtent extent, CancellationToken ct)`:
  1. For each block in extent (StartBlock to StartBlock + BlockCount - 1):
     a. Read data block (full 4096B payload)
     b. Locate corresponding TRLR block: trlr_physical = floor(logicalIndex / 255) * 256 + 255
     c. Read TRLR record (16B): DataBlockTypeTag, GenerationNumber, XxHash64
     d. Compute XxHash64 over data block payload
     e. Compare computed vs stored XxHash64
  2. Return per-block pass/fail results
- `BlockLevelResult`: `int BlocksVerified, int BlocksFailed, IReadOnlyList<long> FailedBlockNumbers`

Level 2 - Extent verification (BLAKE3 ExpectedHash):
- `Task<ExtentLevelResult> VerifyExtentAsync(InodeExtent extent, CancellationToken ct)`:
  1. Read all blocks in extent
  2. Compute BLAKE3 hash over concatenated block data, truncate to 16 bytes
  3. Compare with extent's ExpectedHash field
  4. Return match/mismatch
- `ExtentLevelResult`: `bool HashMatches, byte[] ComputedHash, byte[] StoredHash, long StartBlock, int BlockCount`

Level 3 - File verification (Merkle root):
- `Task<FileLevelResult> VerifyFileAsync(long inodeNumber, IReadOnlyList<InodeExtent> extents, CancellationToken ct)`:
  1. Compute leaf hashes from extent data
  2. Build Merkle tree and compute root
  3. Compare with stored Merkle root from Integrity Tree region (via MerkleIntegrityVerifier)
  4. Return match/mismatch
- `FileLevelResult`: `bool RootMatches, byte[] ComputedRoot, byte[] StoredRoot, int LeafCount`

Full chain verification:
- `Task<ChainVerificationResult> VerifyChainAsync(long inodeNumber, IReadOnlyList<InodeExtent> extents, long dataRegionStart, CancellationToken ct)`:
  1. Run Level 1 (blocks) for all extents
  2. If Level 1 passes (or StopOnFirstFailure == false): Run Level 2 (extent hashes)
  3. If Level 2 passes (or StopOnFirstFailure == false): Run Level 3 (Merkle root)
  4. Aggregate results into ChainVerificationResult

- `ChainVerificationResult` readonly struct:
  - `bool AllLevelsPassed`
  - `IntegrityLevel? FirstFailureLevel` -- null if all passed
  - `BlockLevelResult BlockResult`
  - `IReadOnlyList<ExtentLevelResult> ExtentResults`
  - `FileLevelResult? FileResult`
  - `TimeSpan Duration`
