- Change `public uint Value { get; }` to `public ulong Value { get; }`
- Constructor: accept `ulong value`
- `IsModuleActive`: `(Value &amp; (1UL &lt;&lt; (int)module)) != 0`
- `WithModule`: `new(Value | (1UL &lt;&lt; (int)module))`
- `WithoutModule`: `new(Value &amp; ~(1UL &lt;&lt; (int)module))`
- `ActiveModuleCount`: `BitOperations.PopCount(Value)` (already works with ulong)
- `GetActiveModuleIds`: iterate with `ulong bits` instead of `uint bits`
- `FromModules`: use `ulong value` accumulator with `1UL &lt;&lt;`
- `AllModules`: update mask from `0x0007_FFFFu` to cover all 39 defined modules: `0x0000_007F_FFFF_FFFFuL` (bits 0-38)
- `None`: `new(0UL)`
- Implicit conversions: `ulong` instead of `uint`
- `Serialize`: `BinaryPrimitives.WriteUInt64LittleEndian`, buffer must be >= 8 bytes
- `Deserialize`: `BinaryPrimitives.ReadUInt64LittleEndian`, buffer must be >= 8 bytes
- `GetHashCode`: `Value.GetHashCode()`
- `ToString`: `$"ModuleManifest(0x{Value:X16}, {ActiveModuleCount} active)"`

Update `FormatConstants`:
- `MaxModules = 64` (was 32)
- `DefinedModules = 39` (was 19)
