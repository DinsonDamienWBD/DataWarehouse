Create `VdeWritePipeline.cs` with `[SdkCompatibility("6.0.0")]`:
- Namespace: `DataWarehouse.SDK.VirtualDiskEngine.Pipeline`
- `class VdeWritePipeline`:
  - Constructor: `IReadOnlyList<IVdeWriteStage> stages` -- accepts pre-ordered stages. The caller is responsible for ordering (InodeResolver first, WalBlockWriter last). Store as `_stages`.
  - `async Task<VdePipelineContext> ExecuteAsync(VdePipelineContext context, CancellationToken ct)`:
    1. Set `context.Direction = PipelineDirection.Write`
    2. For each stage in `_stages`:
       a. If `stage.ModuleGate` is not null AND `!context.Manifest.IsModuleActive(stage.ModuleGate.Value)`:
          - Add stage.StageName to `context.StagesSkipped`
          - Continue (skip)
       b. Else:
          - `await stage.ExecuteAsync(context, ct)`
          - Add stage.StageName to `context.StagesExecuted`
    3. Return context
  - `IReadOnlyList<IVdeWriteStage> Stages => _stages` -- expose for introspection
  - Spec reference: Write pipeline stages are: InodeResolver, ModulePopulator, TagIndexer, DataTransformer, ExtentAllocator+RAID, IntegrityCalculator, WAL+BlockWriter (7 stages per AD-53)

Create `VdeReadPipeline.cs` with `[SdkCompatibility("6.0.0")]`:
- Namespace: `DataWarehouse.SDK.VirtualDiskEngine.Pipeline`
- `class VdeReadPipeline`:
  - Constructor: `IReadOnlyList<IVdeReadStage> stages` -- pre-ordered stages
  - `async Task<VdePipelineContext> ExecuteAsync(VdePipelineContext context, CancellationToken ct)`:
    1. Set `context.Direction = PipelineDirection.Read`
    2. Same module-gated loop as write pipeline but with `IVdeReadStage`
    3. Return context
  - `IReadOnlyList<IVdeReadStage> Stages => _stages`
  - Spec reference: Read pipeline stages are: InodeLookup, AccessControl, ExtentResolver, BlockReader+Integrity, ModuleExtractor, PostReadUpdates (6 stages per AD-53)

Both pipelines:
- Check `ct.ThrowIfCancellationRequested()` before each stage
- Are simple orchestrators -- they do NOT implement any stage logic themselves
- Log stage execution to `context.StagesExecuted` / `context.StagesSkipped`
