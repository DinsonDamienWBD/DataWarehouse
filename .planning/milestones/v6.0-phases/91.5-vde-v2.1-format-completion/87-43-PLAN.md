  - `Task<TpmQuote> GenerateQuoteAsync(int[] pcrIndices, byte[] nonce, CancellationToken ct)`
  - `Task<bool> VerifyQuoteAsync(TpmQuote quote, byte[] expectedNonce, CancellationToken ct)`

Proof generation:
- `Task<CustodyProof> GenerateProofAsync(ITpmProvider tpm, long integrityAnchorBlock, Guid volumeUuid, CancellationToken ct)`:
  1. Read Integrity Anchor block (Block 3 of Superblock Group)
  2. Extract MerkleRootHash (32 bytes at offset 0)
  3. Generate nonce: BLAKE3(MerkleRootHash || VolumeUuid || current UTC timestamp)
  4. Request TPM quote with nonce (binds PCR state to VDE state)
  5. Build CustodyProof combining all elements

- `CustodyProof` record:
  - `byte[] MerkleRootHash` -- 32B from Integrity Anchor
  - `Guid VolumeUuid`
  - `DateTimeOffset Timestamp`
  - `byte[] Nonce` -- BLAKE3 binding nonce
  - `TpmQuote TpmQuote` -- signed TPM attestation
  - `byte[] ProofSignature` -- HMAC-BLAKE3 over entire proof structure

Proof verification:
- `Task<CustodyVerificationResult> VerifyProofAsync(CustodyProof proof, ITpmProvider tpm, long integrityAnchorBlock, CancellationToken ct)`:
  1. Read current MerkleRootHash from Integrity Anchor
  2. Verify proof.MerkleRootHash matches current (data unchanged)
  3. Recompute nonce from components, verify matches proof.Nonce
  4. Verify TPM quote signature via tpm.VerifyQuoteAsync
  5. Verify PCR values in quote match expected platform state
  6. Check timestamp within ProofExpiryHours
  7. Return verification result

- `CustodyVerificationResult` readonly struct:
  - `bool IsValid`
  - `bool MerkleRootMatches, bool TpmQuoteValid, bool NonceValid, bool TimestampValid`
  - `string? FailureReason`
  - `TimeSpan ProofAge`

Serialization:
- `byte[] SerializeProof(CustodyProof proof)` -- portable binary format
- `CustodyProof DeserializeProof(ReadOnlySpan<byte> data)` -- reconstruct from bytes
