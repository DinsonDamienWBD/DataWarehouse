Create `VacuumConfig` class with `[SdkCompatibility("6.0.0")]`:
- `TimeSpan SlaLeaseTimeout { get; set; } = TimeSpan.FromSeconds(300)` -- max time a reader can hold an epoch lease
- `TimeSpan VacuumInterval { get; set; } = TimeSpan.FromSeconds(30)` -- background vacuum cycle interval
- `int MaxVersionsPerCycle { get; set; } = 2000` -- cap per cycle to avoid pauses
- `bool WormExemptionEnabled { get; set; } = true` -- never vacuum WORM-flagged objects
- `bool MaterializeLongRunningSnapshots { get; set; } = true` -- copy snapshot data before expiration
- `TimeSpan MaterializationWarningThreshold { get; set; } = TimeSpan.FromSeconds(240)` -- warn 60s before timeout
- `double DeadSpaceRatioThreshold { get; set; } = 0.3` -- trigger aggressive vacuum above this ratio

Create `EpochExpiredException` class: inherits `InvalidOperationException`, `[SdkCompatibility("6.0.0")]`
- Constructor: `long epoch, TimeSpan elapsed, TimeSpan limit`
- Properties: `long ExpiredEpoch, TimeSpan Elapsed, TimeSpan Limit`

Create `EpochTracker` class with `[SdkCompatibility("6.0.0")]`:
- Manages reader epoch registrations with lease enforcement
- `ConcurrentDictionary<long, ReaderLease>` where `ReaderLease` readonly struct: `long ReaderId, long Epoch, DateTimeOffset AcquiredAt, bool IsMaterialized`
- `long CurrentGlobalEpoch { get; }`: monotonically increasing, advanced on each transaction commit
- `long AdvanceEpoch()`: increment and return new epoch (Interlocked.Increment)
- `ReaderLease AcquireReaderLease(long readerId)`: register at current epoch, record timestamp
- `void ReleaseReaderLease(long readerId)`: remove from active set
- `long OldestActiveEpoch { get; }`: min epoch across all active reader leases; if no readers, returns CurrentGlobalEpoch
- `IReadOnlyList<ReaderLease> GetExpiredLeases(TimeSpan timeout)`: returns leases older than timeout
- `void ExpireLease(long readerId)`: forcibly remove expired lease (caller should have materialized or thrown)

Create `EpochBasedVacuum` class with `[SdkCompatibility("6.0.0")]`, implements `IAsyncDisposable`:
- Constructor: `MvccGarbageCollector innerGc, MvccVersionStore versionStore, EpochTracker epochTracker, VacuumConfig config, IBlockAllocator allocator`

Background vacuum loop:
- `Task RunAsync(CancellationToken ct)`:
  1. Check for expired leases. For each expired lease:
     a. If MaterializeLongRunningSnapshots: copy snapshot data to temporary location, mark lease as materialized
     b. If not materializable: throw EpochExpiredException on next read from that reader
     c. Remove expired lease from tracker
  2. Compute safe vacuum epoch = epochTracker.OldestActiveEpoch
  3. Scan version chains for dead versions (TransactionId < safe vacuum epoch)
  4. For each dead version:
     a. Check if inode has WORM flag (InodeFlags bit 2) -- if yes, skip (WORM exemption)
     b. Unlink from version chain
     c. Free blocks via allocator
  5. Update metrics
  6. Sleep VacuumInterval
  7. Respects cancellation

Metrics:
- `VacuumMetrics GetMetrics()` returns readonly struct:
  - `double DeadSpaceRatio` -- fraction of version store occupied by dead versions
  - `double ReclaimRateBytesPerSecond` -- rolling average
  - `long OldestLivingEpoch` -- from epoch tracker
  - `long TotalVersionsReclaimed, long TotalBlocksFreed`
  - `int ActiveReaderCount, int ExpiredLeaseCount`
  - `DateTimeOffset LastVacuumTime, TimeSpan LastVacuumDuration`

Aggressive mode:
- If DeadSpaceRatio > config threshold, reduce VacuumInterval to 5s and double MaxVersionsPerCycle
