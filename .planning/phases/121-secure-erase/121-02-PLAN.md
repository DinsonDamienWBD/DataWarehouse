         - `int Length { get; }` — buffer length
         - `bool IsZeroed { get; }` — true after Dispose
    2. Create `DataWarehouse.SDK/Security/SecureErase/MemoryZeroizationGuard.cs`:
       - `MemoryZeroizationGuard : IMemoryProtector`:
         - Constructor takes `byte[] buffer`
         - `Dispose()` calls `CryptographicOperations.ZeroMemory(buffer)`, sets `IsZeroed = true`
         - `AsSpan()` throws `ObjectDisposedException` if already disposed
         - Thread-safe disposal via `Interlocked.Exchange`

         - Static factory: `Rent(int length)` — rents from `ArrayPool<byte>` with zero-on-return

         - Always zeroes buffer in finally block

       - Test: Double Dispose is safe (no exception)
       - Test: SensitiveMemory.Rent zeroes on Dispose
       - Test: SensitiveMemory.Rent returns buffer to pool
       - Test: UseSensitiveBuffer zeroes stackalloc buffer in finally
