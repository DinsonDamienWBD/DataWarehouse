**EpochRecoveryEngine.cs** — Handles mount, recovery, and consistency:
- Namespace: `DataWarehouse.SDK.VirtualDiskEngine.IO.ObjectStorage`
- Constructor: `EpochRecoveryEngine(IObjectStorageClient objectStorage, S3BlockDeviceOptions options, ILogger<EpochRecoveryEngine>? logger = null)`

**MountAsync method** — `Task<MountResult> MountAsync(CancellationToken ct)`:
1. Try to download manifest from `{options.BucketName}/{options.ManifestKey}` via `GetObjectAsync`.
2. If manifest not found (KeyNotFoundException): this is a fresh volume. Create empty manifest with VdeUuid, BlockSize, BlockCount from options. Upload it. Return `MountResult.FreshMount`.
3. If manifest found: deserialize via `EpochSerializer.DeserializeManifest`. Validate VdeUuid, BlockSize match options. If mismatch, throw `InvalidOperationException` with details.
4. Return `MountResult.ExistingMount(manifest)`.

**RecoverAsync method** — `Task<RecoveryResult> RecoverAsync(EpochManifest manifest, CancellationToken ct)`:
1. Scan S3 for any epoch objects with epoch number > `manifest.LatestConsistentEpoch` using `ListObjectsAsync` with the epoch prefix. These are uncommitted epochs from a crash.
2. For each uncommitted epoch object found (sorted by epoch number):
   a. Download via `GetObjectAsync`
   b. Deserialize via `EpochSerializer.DeserializeEpoch`
   c. Verify checksum. If corrupt, log warning and skip (data loss accepted for uncommitted epochs).
   d. If valid, add to manifest via `WithNewEpoch`. This "recovers" the uncommitted epoch.
3. If any epochs were recovered, upload updated manifest.
4. Return `RecoveryResult` with: `int EpochsRecovered`, `int EpochsSkippedCorrupt`, `EpochManifest FinalManifest`.

**RebuildBlockIndexAsync method** — `Task<Dictionary<long, (long EpochNumber, int IndexInEpoch)>> RebuildBlockIndexAsync(EpochManifest manifest, CancellationToken ct)`:
- Simply returns `manifest.BlockIndex` (already computed by manifest deserialization).
- Exists as a separate method for potential future optimization (lazy loading, on-demand download).

**MountResult** sealed record: `(bool IsFresh, EpochManifest Manifest)`
- Static factories: `FreshMount` (creates empty manifest), `ExistingMount(EpochManifest m)`

**RecoveryResult** sealed record: `(int EpochsRecovered, int EpochsSkippedCorrupt, EpochManifest FinalManifest)`
