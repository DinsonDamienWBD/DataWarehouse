**EpochSerializer.cs** — Binary serialization for epoch objects and JSON serialization for manifests:
- Namespace: `DataWarehouse.SDK.VirtualDiskEngine.IO.ObjectStorage`
- Static class `EpochSerializer`

**Epoch object binary format** (little-endian):
```
[Magic:4 "EPCH"][Version:2][VdeUuid:16][EpochNumber:8][BlockSize:4][EntryCount:4][CreatedUtcTicks:8]
For each entry:
  [BlockNumber:8][DataLength:4][Data:DataLength]
[Checksum:8 (XxHash64 of all bytes before checksum)]
```

Methods:
- `static byte[] SerializeEpoch(EpochObject epoch)` — serialize to binary. Use `System.IO.Hashing.XxHash64` for checksum (same as WAL). Use `BinaryPrimitives.Write*LittleEndian` for all numeric fields. Version = 1.
- `static EpochObject DeserializeEpoch(ReadOnlyMemory<byte> data)` — deserialize from binary. Validate magic bytes "EPCH". Verify checksum. Throw `InvalidDataException` on corruption.
- `static byte[] SerializeManifest(EpochManifest manifest)` — serialize manifest to UTF-8 JSON using `System.Text.Json.JsonSerializer` with camelCase naming. Include all fields except BlockIndex (it's rebuilt from Epochs list on load).
- `static EpochManifest DeserializeManifest(ReadOnlyMemory<byte> data)` — deserialize JSON. Rebuild BlockIndex by iterating Epochs in order and tracking latest write for each block.
- `static ulong ComputeEpochChecksum(IReadOnlyList<EpochBlockEntry> entries)` — compute XxHash64 over all entry block numbers and data.

Constants:
- `const uint EpochMagic = 0x48435045u; // "EPCH" little-endian`
- `const ushort CurrentVersion = 1;`

Apply `[SdkCompatibility("8.0.0")]`.
