**AwsS3ObjectStorageClient.cs**:
- Constructor: `AwsS3ObjectStorageClient(Amazon.S3.IAmazonS3? s3Client = null, string? serviceUrl = null, bool forcePathStyle = false)`
- If s3Client is null, create `new AmazonS3Client()` (uses default credential chain). If serviceUrl provided, configure `AmazonS3Config { ServiceURL, ForcePathStyle }` for MinIO/LocalStack compatibility.
- **PutObjectAsync**: Use `PutObjectRequest` with `InputStream` from `MemoryStream`. Set `ContentType` and `Metadata` headers.
- **GetObjectAsync**: Use `GetObjectRequest`. Read response stream into `byte[]`. Return as `Memory<byte>`.
- **DeleteObjectAsync**: Use `DeleteObjectRequest`. Swallow `AmazonS3Exception` with `NoSuchKey` status (S3 semantics).
- **ObjectExistsAsync**: Use `GetObjectMetadataRequest`. Return true. Catch `AmazonS3Exception` with 404 -> return false.
- **ListObjectsAsync**: Use `ListObjectsV2Request` with `Prefix`. Paginate via ContinuationToken. Map to `ObjectStorageEntry`.
- **GetObjectMetadataAsync**: Use `GetObjectMetadataRequest`. Map headers.
- **CopyObjectAsync**: Use `CopyObjectRequest`.
- **DisposeAsync**: Dispose `_s3Client` if we created it (track `_ownsClient`).

**AzureBlobObjectStorageClient.cs**:
- Constructor: `AzureBlobObjectStorageClient(string connectionString)` or `AzureBlobObjectStorageClient(Azure.Storage.Blobs.BlobServiceClient serviceClient)`
- Map bucket -> container, key -> blob name.
- **PutObjectAsync**: `GetBlobClient(container, key).UploadAsync(BinaryData.FromBytes(data), overwrite: true)`. Set metadata via `BlobHttpHeaders`.
- **GetObjectAsync**: `DownloadContentAsync()`. Return `BinaryData.ToMemory()`.
- **DeleteObjectAsync**: `DeleteIfExistsAsync()`.
- **ObjectExistsAsync**: `ExistsAsync()`.
- **ListObjectsAsync**: `GetBlobsAsync(prefix: prefix)`. Map to `ObjectStorageEntry`.
- **GetObjectMetadataAsync**: `GetPropertiesAsync()`.
- **CopyObjectAsync**: `StartCopyFromUriAsync` from source blob URL.
- **DisposeAsync**: No-op (BlobServiceClient is not disposable).

**GcsObjectStorageClient.cs**:
- Constructor: `GcsObjectStorageClient(Google.Cloud.Storage.V1.StorageClient? client = null)`
- If client is null, create `StorageClient.Create()` (uses default credentials).
- **PutObjectAsync**: `UploadObjectAsync(bucket, key, stream, contentType)`.
- **GetObjectAsync**: `DownloadObjectAsync` into `MemoryStream`.
- **DeleteObjectAsync**: `DeleteObjectAsync`. Catch `Google.GoogleApiException` with 404 -> no-op.
- **ObjectExistsAsync**: Try `GetObjectAsync`, catch 404 -> false.
- **ListObjectsAsync**: `ListObjectsAsync(bucket, prefix)`.
- **GetObjectMetadataAsync**: `GetObjectAsync(bucket, key)` -> map metadata.
- **CopyObjectAsync**: `CopyObjectAsync`.
- **DisposeAsync**: Dispose `_client` if we own it.

All three: Apply `[SdkCompatibility("8.0.0")]`. Use `ILogger<T>?` optional parameter. Wrap all SDK calls in try/catch mapping provider-specific exceptions to `ObjectStorageException` (create this sealed exception class in the same file or IObjectStorageClient.cs if it fits).
