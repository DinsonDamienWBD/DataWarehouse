ScriptParser.cs:
- Parse .dws script files into AST (list of ScriptNode)
- Node types: CommandNode, AssignNode, IfNode, ForNode, WhileNode, TryCatchNode, CommentNode
- Syntax: `$var = value`, `if $condition then ... end`, `for $item in $list do ... end`, `while $condition do ... end`, `try ... catch ... end`
- Commands: bare lines are CLI commands (e.g., `backup create --name daily`)
- String interpolation: `backup create --name "daily-$date"`
- Comments: lines starting with #

ScriptRuntime.cs:
- Maintain variable scope (Dictionary&lt;string, object&gt;)
- Built-in variables: $date, $time, $hostname, $user
- Built-in functions: $len(), $upper(), $lower(), $trim(), $split()
- Execute CommandNode by delegating to CommandExecutor
- Capture command output in $result variable
- Support nested scopes (for/while create child scope)
- Error handling: try/catch captures command failures, $error holds last error message
