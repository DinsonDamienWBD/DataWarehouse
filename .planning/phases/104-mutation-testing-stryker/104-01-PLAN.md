1. Install Stryker.NET as a dotnet tool if not already installed:
   ```
   dotnet tool install -g dotnet-stryker
   ```

2. Create `DataWarehouse.Hardening.Tests/stryker-config.json`:
   ```json
   {
     "stryker-config": {
       "project": "DataWarehouse.SDK.csproj",
       "test-projects": ["DataWarehouse.Hardening.Tests/DataWarehouse.Hardening.Tests.csproj"],
       "reporters": ["html", "json", "markdown"],
       "log-level": "info",
       "concurrency": 4,
       "thresholds": {
         "high": 95,
         "low": 80,
         "break": 70
       },
       "mutate": ["!**/obj/**", "!**/bin/**"],
       "excluded-mutations": [],
       "timeout-ms": 30000
     }
   }
   ```

   NOTE: The `project` field targets DataWarehouse.SDK since that is what the hardening tests reference. Adjust if the hardening tests also reference plugin projects — in that case, run Stryker per-project or use solution-level configuration.

3. Run Stryker from the hardening test project directory:
   ```
   cd DataWarehouse.Hardening.Tests
   dotnet stryker --config-file stryker-config.json --output ../Metadata/stryker-hardening-report/
   ```
