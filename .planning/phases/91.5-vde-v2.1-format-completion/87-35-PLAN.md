Inode scanner:
- `Task<InodeAnalytics> ScanInodesAsync(long inodeTableStartBlock, long inodeCount, CancellationToken ct)`:
  1. Sequentially read inode table blocks (8 inodes per 4KB block at 512B each)
  2. Extract per-inode: size, type, extent count, timestamps, active modules, flags
  3. Aggregate: total files, dirs, total logical size, total allocated size, fragmentation ratio, age distribution, module usage histogram
- `InodeAnalytics` readonly struct:
  - `long TotalFiles, TotalDirectories, TotalSymlinks`
  - `long TotalLogicalBytes, TotalAllocatedBytes`
  - `double AverageExtentsPerFile, double FragmentationRatio`
  - `Dictionary<string, long> ModuleUsageCounts` -- how many inodes use each module
  - `long OldestFileEpoch, long NewestFileEpoch`

TRLR scanner:
- `Task<TrailerAnalytics> ScanTrailersAsync(long dataRegionStartBlock, long dataRegionBlockCount, CancellationToken ct)`:
  1. Calculate TRLR block positions (every 256th block: positions 255, 511, 767, ...)
  2. Read only TRLR blocks (skip data blocks entirely)
  3. For each trailer record: check generation numbers, verify XxHash64 format
  4. Aggregate: generation distribution, suspect corruptions (generation == 0 or checksum == 0)
- `TrailerAnalytics` readonly struct:
  - `long TrailerBlocksScanned, long TrailerRecordsRead`
  - `long SuspectCorruptions` -- records with zero generation or zero checksum
  - `uint MinGeneration, uint MaxGeneration`

Tag scanner:
- `Task<TagAnalytics> ScanInlineTagsAsync(long inodeTableStartBlock, long inodeCount, CancellationToken ct)`:
  1. Read inode table sequentially, extract InlineTagArea (176B at offset 320 per inode)
  2. Parse compact tags: [NamespaceHash:4][NameHash:4][ValueType:1][ValueLen:1][Value<=22B]
  3. Aggregate: tag namespace distribution, most common tags, inodes with overflow tags
- `TagAnalytics` readonly struct:
  - `long InodesWithTags, long TotalInlineTags, long InodesWithOverflow`
  - `Dictionary<uint, long> NamespaceHashCounts` -- tag distribution by namespace

Combined scan:
- `Task<ColdAnalyticsReport> RunFullScanAsync(long inodeTableStart, long inodeCount, long dataRegionStart, long dataRegionBlocks, CancellationToken ct)`:
  - Runs enabled scanners, returns combined report
- `ColdAnalyticsReport` readonly struct: `InodeAnalytics? Inodes, TrailerAnalytics? Trailers, TagAnalytics? Tags, TimeSpan Duration`
