Create `PolymorphicRaidModule` readonly struct with `[SdkCompatibility("6.0.0")]` in namespace `DataWarehouse.SDK.VirtualDiskEngine.Format`:

Per spec: extends RAID module (bit 5) base 4B to 32B in overflow. Layout: [Scheme:1][DataShards:1][ParityShards:1][Flags:1][DeviceMap:28]

- `RaidTopologyScheme Scheme` -- enum: Standard=0, Mirror=1, EC_2_1=2, EC_4_2=3, EC_8_3=4 (matches extent flag bits 9-11 encoding)
- `byte DataShards` -- number of data shards (e.g., 4 for EC_4_2)
- `byte ParityShards` -- number of parity shards (e.g., 2 for EC_4_2)
- `PolymorphicRaidFlags Flags` -- [Flags] enum: None=0, InheritFromVolume=1, AllowDegradation=2, ParityDecayEligible=4
- `byte[] DeviceMap` -- 28-byte device mapping (which physical devices hold shards for this inode). Each byte = device index (0-254), 0xFF = unassigned. Max 28 devices per inode.
- `const int SerializedSize = 32`
- `const byte ModuleBitPosition = 5` -- extends existing RAID module in ModuleManifest
- `bool IsErasureCoded` -- true if Scheme >= EC_2_1
- `int TotalShards` -- DataShards + ParityShards
- `double StorageOverhead` -- ParityShards / (double)DataShards (e.g., 0.5 for EC_4_2)
- `static void Serialize(in PolymorphicRaidModule module, Span<byte> buffer)` -- 32B
- `static PolymorphicRaidModule Deserialize(ReadOnlySpan<byte> buffer)` -- 32B
- `static PolymorphicRaidModule Standard` -- no redundancy default
- `static PolymorphicRaidModule CreateMirror()` -- Mirror scheme
- `static PolymorphicRaidModule CreateErasureCoded(byte dataShards, byte parityShards)` -- RS erasure coding
