Allocation-free FROST verifier (per spec: NOT BouncyCastle):
- `bool VerifyAggregateSignature(ReadOnlySpan<byte> message, QuorumOverflowInode seal)`:
  - Implements Schnorr aggregate signature verification using System.Security.Cryptography (Ed25519 if available in .NET 9+, otherwise manual curve25519 implementation)
  - Verify that at least `Threshold` bits set in SignerBitmap
  - Verify aggregate signature against message using combined public key
  - Zero heap allocations on verification hot path — use stackalloc for intermediates
  - Returns true if signature valid AND quorum threshold met

Write path:
- `Task<QuorumSealResult> SealWriteAsync(long inodeNumber, ReadOnlyMemory<byte> data, int replicationGeneration, QuorumOverflowInode seal, CancellationToken ct)`:
  1. Build signed message: `[InodeNumber:8][DataHash:32][ReplicationGeneration:4][Nonce:8]` = 52 bytes
  2. Verify aggregate signature against message
  3. If verification fails:
     - Reject mode: throw QuorumVerificationException
     - AcceptUnsigned mode: write data without seal, log warning
     - QueueForSealing mode: enqueue for later sealing, write data provisionally
  4. If verification passes:
     - WAL-journal the sealed write
     - Write data blocks to device
     - Write QuorumOverflowInode (79B) to inode's module overflow block
     - Set QUORUM_SEALED flag (InodeFlags bit 6)
  5. Return QuorumSealResult

- `QuorumSealResult` readonly struct: `bool Sealed, QuorumDegradePolicy ActionTaken, int SignersContributed, int ThresholdRequired`
- `QuorumVerificationException` class: inherits `InvalidOperationException`, includes `QuorumOverflowInode FailedSeal, string Reason`

Read-path verification:
- `bool VerifyInodeSeal(long inodeNumber, ReadOnlySpan<byte> dataHash, QuorumOverflowInode seal, int replicationGeneration)`:
  - Reconstruct signed message from parameters
  - Verify aggregate signature
  - Useful for integrity audits and replication conflict resolution

Sealing queue (for QUEUE_FOR_SEALING mode):

- `Task<int> ProcessSealingQueueAsync(Func<byte[], Task<QuorumOverflowInode>> sealProvider, CancellationToken ct)`: process pending items, timeout expired items
