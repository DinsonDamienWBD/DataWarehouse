  1. Create OPJR entry (type=ENCRYPT, totalUnits=totalBlocks/MigrationChunkBlocks, payload stores keySlot as first 4 LE bytes)

  type=REKEY, payload stores oldKeySlot(4B) + newKeySlot(4B)

  1. Checkpoint OPJR with completedUnits=chunksProcessedSoFar

  3. Return checkpoint indicating progress for Superblock update

  Transition to COMPLETED. Caller clears EncryptionMigrationKeySlot to 0xFFFFFFFF and progress to 0.

  Returns INPROGRESS encrypt/decrypt/rekey operations for crash recovery resume.
