Create `InstantCloneEngine` class with `[SdkCompatibility("6.0.0")]`:
- Constructor: `IBlockDevice device, IBlockAllocator allocator, IInodeTable inodeTable, int blockSize`

Core operations:
- `Task<CloneResult> CloneInodeAsync(long sourceInodeNumber, long destinationParentInode, string destinationName, CloneOptions options, CancellationToken ct)`:
  1. Read source inode (ExtendedInode512)
  2. Allocate new inode number
  3. Copy inode metadata (size, type, permissions if configured, inline tags if configured)
  4. Copy all extent pointers from source to destination (identical StartBlock, BlockCount)
  5. Set Shared_COW flag on ALL extent pointers in BOTH source AND destination inodes
  6. Increment block reference counts for all shared blocks
  7. Set timestamps on clone (current time unless PreserveTimestamps)
  8. Write both inodes (source updated with Shared_COW flags, destination new)
  9. If DeepClone and source is directory: recursively clone all children
  10. Return CloneResult

- `CloneResult` readonly struct:
  - `long ClonedInodeNumber, int ExtentsShared, long BlocksShared, long LogicalBytesCloned`
  - `int ChildrenCloned` (for directory clones)
  - `TimeSpan Duration`

Write-time CoW split (invoked by write path, not directly by clone):
- `Task<CowSplitResult> SplitOnWriteAsync(long inodeNumber, int extentIndex, long writeOffsetInExtent, int writeLength, CancellationToken ct)`:
  1. Read extent at extentIndex -- check Shared_COW flag
  2. If not Shared_COW: no split needed, return directly
  3. Allocate new blocks for the extent range being written
  4. Copy existing data from shared blocks to new blocks
  5. Apply the pending write to the new blocks
  6. Update extent pointer: new StartBlock, clear Shared_COW flag (if refcount == 1)
  7. Decrement reference count on old blocks; free if refcount == 0
  8. Return CowSplitResult

- `CowSplitResult` readonly struct:
  - `long OldStartBlock, long NewStartBlock, int BlockCount, bool OldBlocksFreed`
