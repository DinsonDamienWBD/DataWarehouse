    - Constructor: ICloudProvider provider, ScalingPolicy policy, ILogger logger
    - `StartAsync(CancellationToken)`: Begins periodic evaluation loop (configurable interval, default 60s)
    - `StopAsync()`: Stops evaluation loop
    - `EvaluateAsync()`: Single evaluation cycle:
      1. ListManagedResourcesAsync to get all managed resources
      2. GetMetricsAsync for each resource
      3. Aggregate metrics (average CPU, max storage, etc.)
      4. Compare against policy thresholds
      5. If scale-up needed and below MaxInstances and past cooldown: ProvisionVmAsync with PreferSpotInstances tag
      6. If scale-down possible and above MinInstances and idle >IdleMinutes and past cooldown: DeprovisionAsync oldest idle instance
      7. Log all scaling decisions
