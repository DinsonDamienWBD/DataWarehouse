    - block: DisksClient.InsertAsync with sizeGb, type from PerformanceTier ("standard"→pd-standard, "premium"→pd-ssd, "ultra"→pd-extreme), labels. Return disk self-link.
    - object: StorageClient.CreateBucket with name "dw-{guid}", location from Region, labels. Return bucket name.

    **DeprovisionAsync(resourceId):**
    - Parse resource path to determine type (instances/disks/buckets). Call appropriate Delete method.
    - Catch RpcException with StatusCode.NotFound → return false. Return true on success.

    **GetMetricsAsync(resourceId):**
    - Build ListTimeSeriesRequest with filter for resource, metrics: compute.googleapis.com/instance/cpu/utilization, compute.googleapis.com/instance/disk/read_bytes_count, etc.
    - Interval: last 5 minutes. Map to CloudResourceMetrics. Return null if no data.

    **ListManagedResourcesAsync():**
    - InstancesClient.ListAsync with filter "labels.managedby=datawarehouse"
    - DisksClient.ListAsync with same filter
    - StorageClient.ListBuckets with label filter
    - Return combined resource identifiers
