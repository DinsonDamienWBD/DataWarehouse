    - Test 6: ShrinkRefusalException is caught and displayed as user-friendly error message

  </behavior>
  <action>
    Create `VdeResizeCommand.cs` in `DataWarehouse.CLI/Commands/`:

    1. **VdeResizeCommand** class following existing CLI command patterns in VdeCommands.cs:
       - Uses Spectre.Console for rich output
       - Subcommand: `dw volume resize`
       - Arguments:
         - `--grow <size>` OR `--shrink <size>` (mutually exclusive, one required)
         - `--offline` (required for shrink, optional for grow -- grow is always online)
         - `<volume-path>` (positional argument: path to .dwvd file)
       - Size parsing: supports GB, MB, TB, GiB, MiB, TiB suffixes. `ParseSize(string)` returns long bytes.

    2. **Grow flow**:
       a. Open volume file, read superblock to get current TotalBlocks and BlockSize
       b. Calculate newTotalBlocks = currentTotalBlocks + (requestedBytes / blockSize)
       c. Create IResizableBlockDevice (FileResizableBlockDevice or detect raw partition)
       d. Call `resizableDevice.GrowUnderlyingAsync(newTotalBlocks, ct)` first
       e. Create VdeResizeEngine, call `GrowAsync(currentTotalBlocks, newTotalBlocks, ct)`
       f. Display Spectre.Console progress bar updating from OPJR progress
       g. On success: display GrowResult summary table

    3. **Shrink flow**:
       a. Require `--offline` flag, refuse without it: "Shrink requires --offline flag. The volume must be unmounted."
       b. Open volume, read superblock
       c. Calculate newTotalBlocks = requestedBytes / blockSize (target total, NOT delta)
       d. Actually: support both `--shrink 50GB` (shrink BY 50GB) and `--shrink-to 50GB` (shrink TO 50GB). Default `--shrink` means shrink BY.
       e. Create VdeShrinkEngine, call `ShrinkAsync(currentTotalBlocks, newTotalBlocks, ct)`
       f. Catch ShrinkRefusalException: display "Cannot shrink: volume has {used} blocks in use, target size {target} blocks is too small. Free at least {used - target} blocks first."
       g. Display Spectre.Console progress bar
       h. On success: display ShrinkResult summary table

    4. Mark with `[SdkCompatibility("8.0.0", Notes = "Phase 116: CLI resize command (SIO-05)")]`.
