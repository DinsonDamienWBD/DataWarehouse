```csharp
[SdkCompatibility("8.0.0", Notes = "Phase 117: RDMA transport via ibverbs (SIO-06)")]
public sealed class RdmaTransport : KernelBypassTransportBase
{
    // Plugin identity
    public override string Id => "kernel-bypass.rdma";
    public override string Name => "RDMA Transport (ibverbs/NetworkDirect)";
    public override string Version => "8.0.0";
    public override PluginCategory Category => PluginCategory.Infrastructure;

    public override KernelBypassCapability Capabilities =>
        KernelBypassCapability.ZeroCopy | KernelBypassCapability.PreRegisteredMemory | KernelBypassCapability.PollMode;
    public override TransportPreference TransportType => TransportPreference.Rdma;

    // ibverbs handles (Linux)
    private nint _context, _pd, _cq, _qp;
    private bool _initialized;

    public override bool IsAvailable => KernelBypassPlatformDetector.IsRdmaAvailable;

    // ConnectAsync: open device, alloc PD, create CQ, create RC QP, modify QP to RTS
    // SendAsync: ibv_post_send with IBV_WR_SEND, poll CQ for completion
    // ReceiveAsync: ibv_post_recv, poll CQ for IBV_WC_RECV
    // PlatformRegisterMemory: ibv_reg_mr with IBV_ACCESS_LOCAL_WRITE | IBV_ACCESS_REMOTE_READ | IBV_ACCESS_REMOTE_WRITE
    // PlatformDeregisterMemory: ibv_dereg_mr
    // SendRegisteredAsync/ReceiveRegisteredAsync: use pre-registered MR for zero-copy
    // ReadRemoteAsync/WriteRemoteAsync: RDMA READ/WRITE verbs (IBV_WR_RDMA_READ/WRITE)
}
```

Key implementation details:
- QP state machine: RESET -> INIT -> RTR -> RTS via ibv_modify_qp
- CQ polling thread using SpinWait for low-latency completion detection
- Pre-registered memory pool for common buffer sizes (4KB, 64KB, 1MB)
- Connection management: store remote QP number and LID during connect handshake
- Error handling: map ibverbs error codes to .NET exceptions, log via ILogger
- Windows path: delegate to NetworkDirect interop when on Windows
- All ibverbs calls guarded with `if (!RuntimeInformation.IsOSPlatform(OSPlatform.Linux)) throw new PlatformNotSupportedException()`
- Dispose: destroy QP, CQ, deregister all MRs, dealloc PD, close device (reverse order)

Tests use a testable design: inject an `IIbverbsNative` interface (internal, defaulting to real P/Invoke) that tests can mock. Test the state machine, memory registration tracking, statistics, error paths, and graceful unavailable behavior.
