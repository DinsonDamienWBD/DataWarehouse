  <action>
1. Create `NvmeDeviceCapabilities.cs`:
   - `public sealed record NvmeDeviceCapabilities`
   - Properties:
     - `string PciAddress` — BDF address of the controller
     - `uint NamespaceId` — probed namespace ID
     - `uint SectorSize` — bytes per sector (512 or 4096)
     - `ulong TotalSectors` — total sector count
     - `long CapacityBytes` — computed: SectorSize * TotalSectors
     - `uint MetadataSize` — per-sector metadata (0 for non-ZNS)
     - `bool IsZns` — true if command set is ZNS (CSI=2)
     - `uint MaxTransferSize` — max bytes per single I/O command
     - `uint MaxQueueDepth` — recommended queue depth (derived from controller limits)
     - `uint NamespaceCount` — total namespaces on controller
     - `bool SupportsNvmeOf` — true if NVMe-oF (TCP/RDMA) transport detected
     - `NvmeNamespaceFlags Flags` — thin provisioning, deallocate, etc.
   - Mark with `[SdkCompatibility("8.0.0", Notes = "Phase 112: NVMe device capability record")]`

2. Create `NvmeNamespaceFlags` enum (`[Flags]`):
   - `None = 0, ThinProvisioning = 1, Deallocate = 2, FlushSupported = 4, OptimalIoSize = 8`

3. Create `NvmeNamespaceProbe.cs`:
   - `internal static class NvmeNamespaceProbe`
   - `static NvmeDeviceCapabilities? Probe(string pciAddress, uint namespaceId = 1)`:
     - Check `SpdkNativeMethods.IsSupported()`, return null if not
     - Probe controller at pciAddress using same pattern as SpdkBlockDevice.ProbeController
     - Get namespace handle via `NvmeCtrlrGetNs`
     - Query: sector size, sector count, metadata size, max transfer size, flags, CSI
     - Query: number of namespaces via `NvmeCtrlrGetNumNs`
     - Determine ZNS from CSI value (2 = ZNS)
     - Compute recommended queue depth: min(hardware max, 256)
     - Detach controller after probing (this is read-only, not for I/O)
     - Return populated `NvmeDeviceCapabilities`
   - `static BlockDeviceOptions CreateOptimizedOptions(NvmeDeviceCapabilities caps)`:
     - Sets `SpdkPciAddress = caps.PciAddress`
     - Sets `SpdkNamespaceId = caps.NamespaceId`
     - Sets `QueueDepth = (int)caps.MaxQueueDepth`
     - Sets `DirectIo = true`
     - Returns configured `BlockDeviceOptions`
   - Mark with `[SdkCompatibility("8.0.0", Notes = "Phase 112: NVMe namespace capability probing")]`
