2. Create `ZnsSpdkBlockDevice.cs`:
   - `public sealed class ZnsSpdkBlockDevice : IDirectBlockDevice, IAsyncDisposable`
   - **Wraps** (not inherits — SpdkBlockDevice is sealed) an inner `SpdkBlockDevice` via composition
   - Constructor: `ZnsSpdkBlockDevice(string pciAddress, uint namespaceId = 1, int queueDepth = 256)`
     - Creates inner `SpdkBlockDevice`
     - Verifies ZNS via `SpdkNativeMethods.NvmeNsGetCsi` (must return 2)
     - Queries zone size and zone count
     - Throws `InvalidOperationException` if namespace is not ZNS
   - Properties:
     - Delegates `BlockSize`, `BlockCount`, `IsDirectIo`, `AlignmentRequirement`, `MaxBatchSize` to inner device
     - `ulong ZoneSize` — LBAs per zone
     - `uint ZoneCount` — total zones
   - Zone management methods:
     - `Task OpenZoneAsync(ulong zoneLba, CancellationToken ct = default)` — sends ZONE_OPEN via `NvmeZnsZoneManagementSend`
     - `Task CloseZoneAsync(ulong zoneLba, CancellationToken ct = default)` — sends ZONE_CLOSE
     - `Task ResetZoneAsync(ulong zoneLba, CancellationToken ct = default)` — sends ZONE_RESET
     - `Task FinishZoneAsync(ulong zoneLba, CancellationToken ct = default)` — sends ZONE_FINISH
     - Each validates zoneLba is zone-aligned (`zoneLba % ZoneSize == 0`)
   - Zone reporting:
     - `Task<ZoneReport> ReportZonesAsync(ulong startLba = 0, ZoneState? filter = null, CancellationToken ct = default)`
     - Allocates DMA buffer, calls `NvmeZnsReportZones`, parses response header + descriptors
     - Maps SPDK zone descriptor structs to managed `ZoneDescriptor` records
   - Write validation:
     - Override `WriteBlockAsync` to check writes are sequential within the target zone
     - Track per-zone write pointers in a `Dictionary<ulong, ulong>` (zone SLBA -> current WP)
     - Refresh write pointers from device on zone open/reset
   - Delegate all `IBlockDevice`/`IBatchBlockDevice`/`IDirectBlockDevice` methods to inner device
   - `DisposeAsync` disposes inner device
   - Mark with `[SdkCompatibility("8.0.0", Notes = "Phase 112: ZNS SPDK block device")]`
  </action>
