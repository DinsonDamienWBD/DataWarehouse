<task type="auto" tdd="true">
  <name>Task 1: ConstantTimeOperations and CacheObliviousAccess</name>
  <files>DataWarehouse.SDK/Security/Tempest/ConstantTimeOperations.cs</files>
  <behavior>
    - FixedTimeEquals wraps CryptographicOperations.FixedTimeEquals
    - ConstantTimeSelect(condition, a, b) returns a if true, b if false without branching
    - ConstantTimeCopy(condition, source, dest) copies only if condition is true, without branching
    - CacheObliviousAccess.Read(array, index) touches all elements to prevent cache-timing leaks
    - PowerLeveling.ExecuteWithDummy(action, minDuration) pads execution time with dummy ops
  </behavior>
  <action>
    Create `DataWarehouse.SDK/Security/Tempest/ConstantTimeOperations.cs`:
    1. `ConstantTimeOperations` static class:
       - `FixedTimeEquals(ReadOnlySpan<byte> left, ReadOnlySpan<byte> right)` — wraps `CryptographicOperations.FixedTimeEquals`
       - `ConstantTimeSelect(bool condition, byte a, byte b)` — `return (byte)((a & mask) | (b & ~mask))` where mask derived from condition without branch
       - `ConstantTimeSelect(bool condition, int a, int b)` — same pattern for int
       - `ConstantTimeCopy(bool condition, ReadOnlySpan<byte> source, Span<byte> dest)` — copies via bitmask, not branch
       - `ConstantTimeCompare(ReadOnlySpan<byte> left, ReadOnlySpan<byte> right)` — returns 0 if equal, non-zero otherwise, in constant time
    2. `CacheObliviousAccess` static class:
       - `Read<T>(T[] array, int targetIndex)` — reads ALL elements, returns only target. Prevents cache-timing analysis of which index was accessed. Use volatile reads to prevent optimizer elimination.
       - `Write<T>(T[] array, int targetIndex, T value)` — writes dummy to all slots, real value to target
       - `Shuffle<T>(T[] array)` — Fisher-Yates with `RandomNumberGenerator` for cryptographic randomness
    3. `PowerLeveling` static class:
       - `ExecuteWithDummy(Action action, TimeSpan minDuration)` — executes action, pads remaining time with dummy crypto operations (SHA-256 hashing of random data) to normalize power consumption
       - `ExecuteWithDummy<T>(Func<T> func, TimeSpan minDuration)` — same with return value
    Mark with `[SdkCompatibility("8.0.0", Notes = "Phase 122: TEMPEST constant-time operations")]`.
  </action>

  </verify>
  <done>ConstantTimeOperations, CacheObliviousAccess, and PowerLeveling classes exist with branch-free operations.</done>
