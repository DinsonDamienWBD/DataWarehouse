<objective>
Implement a per-transaction arena allocator where all allocations during a VDE transaction are freed in O(1) on commit/rollback.

Purpose: ADV-05 requires arena allocators for transactions. VDE transactions accumulate many small allocations (WAL entries, index updates, block copies). Arena allocation avoids per-object GC tracking. On commit/rollback, the entire arena is freed instantly by resetting the pointer — no individual deallocation needed.

Output: IArenaAllocator contract, ArenaAllocator, tests.
</objective>

1. Create `IArenaAllocator` in `DataWarehouse.SDK/Contracts/Memory/`:
   ```
   public interface IArenaAllocator : IDisposable
   {
       unsafe byte* Allocate(int size);
       Span<byte> AllocateSpan(int size);
       void Reset(); // O(1) free all
       ArenaStats GetStats();
   }
   ```
   `ArenaStats`: TotalAllocated, ChunkCount, WastedBytes

2. Create `ArenaAllocator` in `Infrastructure/Memory/`:
   - **Chunk structure**: linked list of native memory chunks
     - Each chunk: `NativeMemory.AlignedAlloc(chunkSize, 4096)` where default chunkSize=1MB
     - Chunk header (in managed code): pointer, size, offset (current allocation point), next chunk ref
   - **Allocate(size)**:
     - If current chunk has room (offset + size <= chunkSize): bump offset, return pointer at old offset
     - Else: allocate new chunk (max(chunkSize, size) for oversized allocations), link to chain, reset offset
     - Alignment: round up size to 8-byte boundary for pointer alignment
   - **Reset()**:
     - Walk chunk chain, `NativeMemory.AlignedFree` each chunk
     - Reset head to null, allocated count to 0
     - This is O(N) in chunks but O(1) amortized per allocation — typically 1-3 chunks per transaction
     - Alternative: keep first chunk, just reset offset (recycle chunk for next txn). Free extra chunks.
   - **ArenaScope**: `IDisposable` wrapper that calls `Reset()` on dispose. Used via `using var scope = arena.CreateScope();`

3. **Usage pattern** (documented in comments, not wired here):
   ```
   using var arena = new ArenaAllocator(chunkSize: 1_048_576);
   using var scope = arena.CreateScope();
   var walEntry = arena.AllocateSpan(128);
   var indexNode = arena.AllocateSpan(64);
   // ... transaction work ...
   // scope.Dispose() -> arena.Reset() -> all memory freed instantly
   ```
