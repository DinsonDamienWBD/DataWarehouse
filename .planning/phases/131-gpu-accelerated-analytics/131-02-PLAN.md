   - `DecryptBatchAsync(ReadOnlyMemory<byte>[] cipherBlocks, byte[] key, byte[] iv, CancellationToken ct) -> Memory<byte>[]`

   - CPU fallback: use K4os.Compression.LZ4 (already in solution) with `Parallel.ForEachAsync`
   - Verify round-trip: `Decompress(Compress(data)) == data` for both GPU and CPU paths
