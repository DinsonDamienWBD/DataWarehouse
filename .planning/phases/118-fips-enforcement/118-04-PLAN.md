   - `static FipsEnforcementMode ResolveFipsMode(DataWarehouseConfiguration config, IReadOnlyList<string> activeComplianceFrameworks)`:
     * If `activeComplianceFrameworks` contains "CJIS", "FedRAMP", "HIPAA", "ITAR", "FedRAMP High", "NIST SP 800-171" -> return `Strict`
     * Else if `config.Security.FipsMode.Value` is true -> return `Strict`

     * Others -> `Disabled`

   - `static IFipsModeEnforcer CreateEnforcer(DataWarehouseConfiguration config, IReadOnlyList<string>? activeComplianceFrameworks = null)`:
     * Resolves mode via `ResolveFipsMode`, creates `FipsModeEnforcer(mode)`

   - `static FipsValidationResult ValidateCipherPresetForFips(CipherPreset preset, IFipsModeEnforcer enforcer, ICipherPresetProvider presetProvider)`:
     * Gets preset config, validates storageCipher + transitCipher + KDF + hash against enforcer
     * Returns first non-approved result or approved result

   - `static IReadOnlyList<CipherPreset> GetFipsCompatiblePresets()`:
     * Returns presets whose default algorithms are all FIPS-approved: FipsOnly, FedRamp, PciDss, Hipaa (these use AES-256-GCM + PBKDF2-SHA256 or AES-256-GCM + Argon2id)
     * Note: Hipaa uses Argon2id which is NOT FIPS — so actually only FipsOnly, FedRamp, PciDss are compatible. Document this finding.

   - `static IReadOnlyList<string> GetFipsTriggeringFrameworks()`:
     * Returns: "CJIS", "FedRAMP", "FedRAMP High", "HIPAA", "ITAR", "NIST SP 800-171"
