- Class ShardRepairService:
  - Constructor(ErasureCodedExtentManager manager, TimeSpan heartbeatInterval)
  - CheckShardHealth(ErasureCodedExtent extent, Func<ShardLocation, Task<bool>> heartbeat): Pings each shard location. Marks unavailable shards. Returns list of unhealthy shards.
  - RepairUnhealthyShards(ErasureCodedExtent extent, ...): For each unhealthy shard, calls manager.RepairExtent. Selects new placement from healthy VDEs using GeoPlacementPolicy. Updates extent metadata with new shard locations.

  - BackgroundRepairLoop: async Task that runs RepairCycle every heartbeatInterval. Implements CancellationToken for graceful shutdown.
