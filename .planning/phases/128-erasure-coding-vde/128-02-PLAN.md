Create ErasureCodedExtentManager.cs:
- Constructor(IErasureCoder coder, IShardPlacementPolicy placementPolicy)
- DistributeExtent(byte[] extentData, int k, int m, IReadOnlyList<Guid> availableVdes): Encodes data into k+m shards. Uses placementPolicy to assign each shard to a VDE (round-robin by default, configurable). Returns ErasureCodedExtent with metadata populated.
- ReconstructExtent(ErasureCodedExtent extent, Func<ShardLocation, Task<byte[]>> shardReader): Reads k available shards via shardReader delegate. Decodes using IErasureCoder. Returns original data.
- VerifyExtent(ErasureCodedExtent extent, Func<ShardLocation, Task<byte[]>> shardReader): Reads all available shards, calls IErasureCoder.Verify.
- RepairExtent(ErasureCodedExtent extent, Func<ShardLocation, Task<byte[]>> shardReader, Func<Guid, byte[], long, Task> shardWriter): Detects unavailable shards. Reconstructs from k available. Writes repaired shards to new locations (from placementPolicy). Updates metadata.
- IShardPlacementPolicy interface: ChoosePlacements(int shardCount, IReadOnlyList<Guid> availableVdes) -> Guid[]
- RoundRobinPlacementPolicy: simple round-robin across available VDEs
